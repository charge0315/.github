# .github リポジトリ

このリポジトリは、charge0315のすべてのパブリックリポジトリに適用されるコミュニティヘルスファイルとGitHubプロファイル設定を含む特別な `.github` リポジトリです。

## 含まれるファイル

- **CODE_OF_CONDUCT.md** - コミュニティ行動規範
- **CONTRIBUTING.md** - コントリビューションガイドライン
- **SECURITY.md** - セキュリティポリシー
- **SUPPORT.md** - サポート情報
- **FUNDING.yml** - スポンサーシップ設定
- **.github/ISSUE_TEMPLATE/** - Issueテンプレート
- **profile/README.md** - GitHubプロファイルページ
